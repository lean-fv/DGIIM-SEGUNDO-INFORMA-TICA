Práctica 1. Imágenes

CREACIÓN DE SUBIMAGEN:

Método que dados ciertos parámetros toma de una imagen original, una subimagen o recorte de dimensiones a especificar.

subimagen <FichImagenOriginal> <FichImagenDestino> <fila> <columna> <fils_sub> <cols_sub>

Parameters
<FichImagenOriginal>Fichero donde se almacena la imagen original.
<FichImagenDestino>Fichero donde se almacena la subimagen resultado.
<fila>Posición fila desde donde empezar a extraer.
<columna>Posición columna desde donde empezar a extraer.
<fils_sub>Alto de la submatriz.
<cols_sub>Ancho de la submatriz.
