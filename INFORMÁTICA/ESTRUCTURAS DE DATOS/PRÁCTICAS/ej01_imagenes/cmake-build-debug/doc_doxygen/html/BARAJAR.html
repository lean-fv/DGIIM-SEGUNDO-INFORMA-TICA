BARAJADO DE FILAS Y ANÁLISIS DE EFICIENCIA

Altera el orden original de filas mediante un algoritmo basado en números coprimos, de forma que la nueva fila r_n = (r*p) mod rows, donde r es el número de fila, p es un número coprimo de r, y rows es el total de filas.

barajar <FichImagenOriginal> <FichImagenDestino>

Parameters
<FichImagenOriginal>Fichero donde se almacena la imagen original.
<FichImagenDestino>Fichero donde se almacena la imagen resultado.

Atendiendo a la eficiencia teórica:

  • Para una implementación de bloques contiguos en memoria:
    • Si las filas y columnas varían, claramente obtenemos una eficiencia de O(rows*cols) al recorrer la matriz entera.
    • Si las filas o columnas están fijas, en ambos casos obtendremos una efciencia de 0(rows) y O(cols), respectivamente.
  • Para una implementación de bloques no contiguos en memoria:
    • Tanto si las filas y columnas varían, como si las columnas permanecen fijas obtenemos una eficiencia de O(rows) al tener que intercambiar solo filas.
    • Si las filas permanecen fijas tendremos una eficiencia de O(1).
